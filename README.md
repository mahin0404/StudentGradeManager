# Student Grade Manager
Student Grade Manager is a Java desktop application built using Java Swing. It allows users to add students with their grades, automatically calculates the average, and assigns a letter grade (A, B, C, D, or F). Students can also be removed from the list at any time.
Concepts Implemented
1. Event Handling Buttons like "Add Student" and "Remove Selected" respond to user clicks. Pressing Enter in the grades field also triggers the add action.
2. Exception Handling If the user enters an empty name, letters in the grade field, or a grade outside 0–100, a clear error message is shown instead of crashing the program.
3. Code Refactoring The code is divided into small focused methods like validateName(), validateGrades(), calcAverage(), and getLetterGrade() so no code is repeated.
4. Unit Testing Six test cases run automatically when the program starts. Results are printed in the console showing PASS or FAIL for each test.
5. Git & GitHub The project is maintained using Git with meaningful commit messages and pushed to a GitHub repository for version control.
